# pyJazz #

pyJazz is a Python port of the amazing JavaScript testing framework [Jasmine](https://jasmine.github.io/).

```py
class MyArithmetic(Describe):
  def it_should_add_2_and_2(self):
    expect(2 + 2).toEqual(4)
```
